Navigation

We are going to create the navigation that accesses to other pages. There are two types of navigation: Graphical and Text.

Making Buttons on Adobe Fireworks

We used Adobe Fireworks to create navigation buttons.

First, we made rectangle boxes for the links, and put any words using text tool.

Next, select all box and text, right click, and choose "Convert to Symbol."

Select "Button" to save the box as a button symbol.

After we create the symbol, put the filter into "Inner Bevel" so that the flat box has a texture of the button.
