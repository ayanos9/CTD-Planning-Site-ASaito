Fireworks/Dreamweaver

What are the strengths/weaknesses of using fireworks and dreamweaver when you creeate webpage?

Fireworks

Strengths

  • Since most of parts are made of pictures, it is easy to change size.
  • We can create layers - Easy to categorize parts.

Weaknesses

  • Hard to update news/events - We need to edit in png and export in html again.

Dreamweaver

Strengths

  • There is html codes - We can also edit html text.
  • No layers

Weaknesses

  • Difficult to change the location of picture.
